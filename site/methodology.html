Unified Scoring Formula

All uORF candidates (AUG and near-cognate) are scored on the same 0-10 scale using five weighted components. This allows direct comparison across target types and incorporates practical ASO designability.

ComponentWeightLogic
Canonical membership3.0 (max)1.0 if in canonical transcript, 0.33 if shared with canonical UTR, 0 if non-canonical only. Near-cognate uORFs scanned from canonical UTR get 1.0.
Ribo-seq evidence2.0 (max)AUG uORFs: ribo_sum / CDS_control_sum, capped at 1.0. Near-cognate: (peak_mean × enrichment) / CDS_control_sum, capped at 1.0.
Start codon strength1.0 (max)Codon-specific initiation efficiency. AUG = 1.0 (modulated by Kozak: 0.5 + 0.5 × kozak/3). CTG/CUG = 0.5. GTG/GUG = 0.4. ACG = 0.3. Others ≤ 0.25.
Conservation1.0 (max)min(|phyloP|, 3.0) / 3.0 at the uAUG ±15 bp window
ASO designability1.0 (max)Fraction of clean ASO candidates (no flags) out of all candidates designed for this position. 1.0 = all clean, 0 = all flagged. Penalizes targets in high-GC regions with homopolymer runs or G4 risk.

composite = (3 × canonical + 2 × ribo_signal + 1 × codon_strength + 1 × conservation + 1 × designability) / 8 × 10

CodonStrengthNotes

This unified formula lets AUG and near-cognate uORFs compete on a level playing field. AUG uORFs benefit from higher start codon strength (up to 1.0 with Kozak 3), while near-cognate candidates can compensate with stronger conservation or Ribo-seq evidence.

Pipeline Steps

Step 1: Transcript Inventory (Ensembl REST)

Query Ensembl REST API for RAI1 (ENSG00000108557) and RAI1-AS1 (ENSG00000237328). Extract all protein-coding transcripts with coordinates, biotype, canonical status, and annotation source (ensembl_havana, havana, havana_tagene).

Source: rest.ensembl.org

Step 2: uORF Discovery

For each protein-coding transcript with a 5′UTR ≥10 bp: extract UTR sequence from exon coordinates, scan for all ATG start codons, translate in-frame until stop (TAA/TAG/TGA) or end of UTR. Record uAUG position, uORF length, and whether it overlaps the main AUG region.

Kozak scoring: +2 if position -3 relative to AUG is A or G; +1 if position +4 is G. Maximum score: 3.

Step 2b: Canonical Triage

For each uORF: check if it was discovered in the canonical transcript (ENST00000353383 / NM_030665.4). If not, map its genomic uAUG position and check whether it falls within the canonical transcript's 5′UTR exons. Classify as: canonical, shared_with_canonical, or non-canonical_only.

Also annotate transcript source quality: ensembl_havana (high), havana (medium), havana_tagene (low).

Step 2c: Ribo-seq Reality Check

Query the GWIPS-viz aggregate ribosome profiling bigWig track (hosted on UCSC as gwipsvizRiboseq) at each uAUG position ±50 bp. Include CDS start as positive control and intron as negative control.

Source: UCSC gwipsvizRiboseq track

Step 3: Conservation Scoring

For each uORF candidate, query UCSC phyloP100way bigWig for three regions: uAUG window (±15 bp), uORF body, and main start neighborhood (last 60 nt of UTR). Report mean phyloP score per region.

Positive phyloP indicates evolutionary conservation (purifying selection). Scores >1.0 suggest strong functional constraint.

Source: UCSC phyloP100way

Steps 4-6: NAT, Expression, QTL

NAT overlap: Compute RAI1-AS1 overlap with RAI1 promoter region, exons, and 5′UTR exons. Score quality on a 0-4.5 scale.

Expression: Scrape tissue expression from Human Protein Atlas for RAI1 brain regions.

eQTL/sQTL: Query GTEx API v8/v10. RAI1 has zero significant QTLs in any tissue, confirming tight transcriptional control.

Step 8: Non-AUG uORF Scan

Scan the canonical transcript's 5′UTR for uORFs initiated by near-cognate start codons: CTG/CUG, GTG/GUG, ACG, TTG/UUG, AAG, AGG, ATC, ATT, ATA. Require at least 2 amino acids and an in-frame stop codon.

Ribo-seq validation: For each candidate, query GWIPS-viz ribosome profiling signal in a tight window (±5 bp around the start codon) and compare to a wider background window (±50 bp). Only candidates with >2x enrichment of peak signal over local background are retained. This filters out scanning signal (ribosomes passing through) from genuine initiation events.

Conservation: phyloP100way scores queried at the ±15 bp window around each near-cognate start codon, using the same UCSC bigWig as for AUG uORFs.

Step 10: RNA Secondary Structure

Compute local RNA secondary structure across the canonical 5′UTR using ViennaRNA (RNAfold). A sliding window of 60 nt with 10 nt step size scans the entire UTR. For each window, the minimum free energy (MFE) of the optimal fold is computed.

Regions with highly negative MFE (stable structures) are predicted to impede ribosome scanning. ASOs that hybridize to these regions can melt the structure, increasing ribosome throughput and translation of the downstream main ORF.

Scoring: Windows are ranked by MFE (most negative = most stable = best structure-melting ASO target). MFE per nucleotide is reported to normalize for GC content effects.

Tool: ViennaRNA 2.7.2 (RNA.fold). Note: this is a thermodynamic prediction, not experimentally validated. SHAPE-seq or DMS-seq data would confirm the structures in vivo.

Step 11: miRNA Target Sites

Conserved miRNA binding sites in the RAI1 3′UTR are retrieved from TargetScan 8.0 (Summary_Counts.default_predictions.txt, filtered for gene symbol RAI1).

Only conserved sites (present across mammalian species) are reported. Each miRNA family is scored by the cumulative weighted context++ score (more negative = stronger predicted repression). The aggregate PCT (probability of conserved targeting) provides an independent measure of functional relevance.

Therapeutic implication: Blocking a miRNA binding site with an ASO would prevent miRNA-mediated mRNA degradation or translational repression, increasing RAI1 mRNA stability and/or translation. This mechanism stacks with 5′UTR-based approaches (uORF blocking, structure melting) since it acts on mRNA abundance rather than translation efficiency.

Step 12: G-Quadruplex Scan

The canonical 5′UTR is scanned for G-quadruplex motifs using the consensus pattern G3+N1-7G3+N1-7G3+N1-7G3+ (and the C-rich complement for i-motifs). G-quadruplexes are extremely stable non-canonical structures that block ribosome scanning more effectively than Watson-Crick hairpins.

Result for RAI1: No G4 motifs found despite 78% GC content. The GC richness is distributed across the UTR rather than concentrated in the clustered G-runs required for G4 formation. This rules out G4-resolving ASOs or small molecules (e.g., PDS, TMPyP4) as a therapeutic strategy for RAI1.

Step 13: 3′UTR Secondary Structure

The canonical 3′UTR (1,472 bp) is analyzed for RNA secondary structure using ViennaRNA (RNAfold) with 80 bp sliding windows and 20 bp step size. Stable 3′UTR structures can affect mRNA stability, subcellular localization, and polyadenylation efficiency.

Therapeutic relevance: ASOs that disrupt stable 3′UTR structures could alter mRNA half-life or accessibility to the translation/degradation machinery. This is a distinct mechanism from 5′UTR-based approaches and could stack with them.

Step 14: RNA-Binding Protein Sites

ENCODE ChIP-seq clustered transcription factor binding sites (encRegTfbsClustered track on UCSC) are queried across the full RAI1 gene region. Sites are classified by location: 5′UTR/promoter region (first 1 kb), 3′UTR region (last 2 kb), or gene body.

Limitation: This track contains ChIP-seq data for transcription factors, not eCLIP data for RNA-binding proteins. True RBP binding sites (e.g., from ENCODE eCLIP experiments) would require querying individual eCLIP tracks, which is a future improvement. The current data is most informative for the promoter/5′UTR region where TF binding affects transcription initiation.

Step 15: Alternative Polyadenylation

The 3′UTR sequence is scanned for canonical (AATAAA) and 10 variant polyadenylation signals. Each signal is classified by distance from the annotated transcript end: signals within 50 bp of the end are marked as "likely active."

Relevance: If RAI1 uses alternative polyA sites, different 3′UTR lengths would have different miRNA site content and structural properties. Shorter isoforms (using an upstream polyA site) would escape regulation by miRNA sites and structures located downstream of the alternative site.

Result for RAI1: The canonical AATAAA signal is at position 1,443 (29 bp from end), consistent with being the primary active site. One upstream variant (AGTAAA at position 1,397) could represent a minor alternative site producing a 3′UTR ~75 bp shorter.

Step 16: ASO Candidate Design

For each target position (all 12 uORFs + structure hotspots), candidate ASO sequences are designed by extracting the mRNA target region at multiple lengths (16, 18, 20 nt) and offsets (±4 nt), computing the reverse complement, and scoring each candidate.

Scoring (ViennaRNA): Each ASO is scored by: (1) duplex binding MFE (ASO-target, more negative = stronger), (2) self-folding MFE (ASO alone, less negative = better), and (3) GC content penalty (optimal ~50-55%).

Quality formula: quality = (-duplex_mfe / length) - gc_penalty - self_fold_penalty

Flags: Candidates are flagged for: high_GC (>75%), self_folding (MFE < -5 kcal/mol), homopolymer (≥5 identical bases), G4_risk (GGGG or CCCC run). Unflagged candidates are labeled "clean."

Designability score: The fraction of clean candidates for each target position. This is the 5th component of the composite score (weight 1/8). Targets with zero clean candidates are penalized but not eliminated, as modified ASO chemistries may overcome design challenges.

Chemistry: Steric blocking ASOs (fully modified 2′-O-Me or LNA/2′-O-Me mixmer, phosphorothioate backbone). NOT gapmers, which would recruit RNase H and degrade the mRNA.

Important: Off-target analysis (BLAST against the transcriptome) is NOT included in this pipeline. It is a required step before experimental use.

Step Numbering Note

The pipeline has 16 steps. Steps 1-6 cover initial data gathering (transcripts, uORFs, triage, Ribo-seq, conservation, NAT overlap, expression, eQTL). Step 7 generates the decision memo. Steps 8-16 are extended analyses added iteratively: non-AUG uORFs (8), NMD/poison exons (9), 5′UTR structure (10), miRNA (11), G-quadruplex (12), 3′UTR structure (13), RBP sites (14), polyadenylation (15), and ASO design (16). Steps 7 and 9 are not shown separately above because step 7 is a report generator (not an analysis) and step 9 results are displayed in the NMD section on the dashboard.

IRES Risk Assessment

The canonical 5′UTR is assessed for features associated with internal ribosome entry site (IRES) activity. If IRES-mediated translation is the dominant mechanism for RAI1, uORF blocking would be ineffective because ribosomes would enter downstream of the uORFs.

Features scored: UTR length (>150 bp), GC content (>60%), polypyrimidine tracts (≥8 nt), structural stability (<-0.4 kcal/mol/nt), and cap-to-AUG distance (>200 nt). RAI1 scores 5/5. No experimentally validated IRES activity has been reported for RAI1 (IRESite database, PubMed search). The feature overlap between IRES-containing UTRs and difficult-to-scan UTRs means this score is a risk flag, not a diagnosis.

Experimental resolution: A bicistronic reporter assay (RAI1 5′UTR between two cistrons, with and without functional cap) would definitively determine whether IRES activity is present.

Limitations

  • uORFs are computationally predicted, not experimentally validated. Reporter assays are needed to confirm functional impact on RAI1 translation.
  • Ribo-seq signal is from the GWIPS-viz aggregate track (multiple studies, multiple cell types). RAI1-specific cell-type Ribo-seq would be more informative.
  • Conservation scores may be inflated for non-canonical transcripts if their UTR regions overlap coding exons of other isoforms.
  • The composite score weights are heuristic, not statistically derived. They reflect our prior that canonical transcript membership is the strongest predictor of therapeutic relevance.
  • ASO design requires experimental optimization (chemistry, length, modification pattern). The target windows here are starting points, not final designs.
  • RNA secondary structure predictions are thermodynamic (MFE), not experimentally validated. In vivo structures may differ due to RNA-binding proteins and co-transcriptional folding. SHAPE-seq or DMS-seq would provide experimental validation.
  • miRNA target predictions are computational (TargetScan context++ model). Experimental validation (luciferase reporter with miRNA site mutations, or AGO-CLIP data) would confirm functional repression.
  • Near-cognate uORF initiation is harder to validate than AUG uORFs. Ribosome profiling enrichment suggests initiation but does not prove it. Mass spectrometry of the predicted peptides would be confirmatory.
  • The Ribo-seq metric differs between AUG uORFs (absolute sum / CDS control) and near-cognate uORFs (peak enrichment over local background / CDS control). Both are normalized to 0-1 for scoring, but the underlying measurements are not directly comparable. Targets of different types should be compared cautiously.
  • Haploinsufficiency ceiling: only one RAI1 allele is available for upregulation. Maximum theoretical increase is 2x from baseline (~50% to ~100% of normal). uORF derepression typically achieves partial, not complete, derepression. The magnitude of upregulation achievable by ASO-mediated uORF blocking is unknown for RAI1.
  • Composite scores within ~1 point of each other should be considered effectively equivalent. The ranking provides suggested priority, not definitive ordering.
  • The top-ranked miRNA (miR-144-3p) is primarily expressed in hematopoietic cells, not neurons. Its relevance to RAI1 regulation in the brain is uncertain.
  • The G-quadruplex scan uses the canonical consensus motif (G3+N1-7G3+). Non-canonical G4 structures (bulged, two-quartet) would not be detected and cannot be ruled out.
